**来源：** [arXiv:1404.3274](https://arxiv.org/pdf/1404.3274) (Lempérière, Deremble, Seager, Potters, Bouchaud - Capital Fund Management, 2014)

**日期：** 2026-05-22（入库）

## 策略框架

CFM提出的**波动率归一化时间序列动量（volatility-normalized time series momentum）**：

### 信号定义

```
s_n(t) = [p(t-1) - EMA_p_n(t-1)] / σ_n(t-1)
```

其中：
- EMA_p_n = 过去价格的指数移动平均（衰减周期 = n 个月）
- σ_n = 价格绝对变化的指数移动平均（same decay）
- n = 2~20个月，最优在3~7个月

### 交易规则

- 信号 > 0 → **做多**
- 信号 < 0 → **做空**
- 波动率归一化仓位：每单位波动率投入固定风险

### 表现（1960~2014）

| 指标 | 值 |
|------|------|
| Sharpe（全品种组合） | 0.78 |
| t-stat | 5.9 |
| 最优化周期 | n=3~7个月 |
| 商品期货 Sharpe | 0.80 |
| 2000年后 Sharpe | 0.75（仍然显著） |

## 可实现的策略方向

### 方向1：月频EMA偏离趋势信号 ✅（主要方向）
- 日线/4小时K线，计算月度EMA偏离
- 波动率归一化信号 → 决定多空方向
- 固定手数仓位或波动率目标仓位

### 方向2：多周期EMA信号组合
- 同时跟踪3/5/10个月信号
- 多数信号同向时才入场（共识过滤）
- 减少假信号

### 方向3：CFM信号 + ATR止损
- 原始信号无独立止损（信号反转即离场）
- 添加ATR移动止损以减少回撤

## 与已有策略的差异

| 维度 | CFM信号策略 | MultiMaTrend | Donchian/Turtle |
|------|-----------|-------------|-----------------|
| 信号类型 | 波动率归一化EMA偏离 | 简单MA交叉权重 | 通道突破 |
| 仓位管理 | 波动率归一化 | 固定手数 | ATR百分比 |
| 持有周期 | 月频（2~20个月） | 日频 | 日频~周频 |
| 离场机制 | 信号反转 | 信号反转 | 反突破/ATR止损 |
