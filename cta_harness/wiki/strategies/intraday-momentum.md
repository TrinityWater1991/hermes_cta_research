# IntradayMomentumStrategy — 日内区间突破策略

**状态**: 已产出（Retain，可用但未达毕业标准）

**策略文件**: `strategies/intraday_momentum_strategy.py`

---

## 设计思路

基于 Opening Range Breakout（开盘区间突破）模式，结合日内动量学术研究：

1. 日盘开盘后前55分钟形成"开盘区间"（记录期间最高点和最低点）
2. 观察窗口结束后，计算突破价位：上轨 = 区间高点 + 区间宽度 × multiplier，下轨 = 区间低点 - 区间宽度 × multiplier
3. 价格突破上轨做多，突破下轨做空
4. 14:45 强制平仓，不隔夜持仓
5. ATR移动止损保护

## 关键技术特征

- **K线周期**: 1分钟（原始数据级别，直接在on_bar执行逻辑）
- **入场逻辑**: 开盘区间突破（仅日盘交易，夜盘不参与）
- **离场逻辑**: 尾盘强制平仓 + ATR移动止损
- **止损**: 6.5倍ATR移动止损
- **日内交易**: 每日最多一次交易（`traded_today` 标记控制）

## 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| observation_minutes | 55 | 开盘观察窗口（分钟） |
| range_multiplier | 1.0 | 区间宽度倍数 |
| atr_window | 20 | ATR计算周期 |
| atr_stop_multiplier | 6.5 | ATR止损倍数 |
| fixed_size | 1 | 固定交易手数 |
| exit_hour | 14 | 强制平仓小时 |
| exit_minute | 45 | 强制平仓分钟 |

## 评估

- **优点**: 低回撤（仅-2.8%），两段分期测试均为正收益
- **不足**: Sharpe未达毕业标准（1.0），2021年后效应明显衰减
- **关键发现**:
  1. 纯方向信号（动量/反转）在rb99上无效
  2. 区间突破模式有效，但需要宽松止损
  3. 日内效应在2021年后衰减，可能与市场结构变化有关
- **结论**: 保留代码，标注为"可用但未达标"策略
