# MultiMaTrendStrategy — 多周期MA趋势因子策略

**状态**: 已产出（Retain，可用但未达毕业标准）

**策略文件**: `strategies/multi_ma_trend_strategy.py`

---

## 设计思路

基于 Han, Zhou, Zhu (2016) "A Trend Factor" 论文思路，将横截面多周期MA因子适配为单标的CTA趋势跟踪：

1. 计算5个周期（10/20/50/100/200）的简单移动平均线
2. 将每个MA归一化为 `(Price - MA) / Price`，正值表示价格在MA上方（上升趋势）
3. 5个信号等权求和，得到综合趋势因子
4. 趋势因子从下方穿越 `entry_threshold` 时做多，从上方穿越 `-entry_threshold` 时做空
5. 趋势因子回归零轴附近（低于 `exit_threshold`）时离场
6. ATR移动止损作为后备保护

## 关键技术特征

- **K线周期**: 1小时（由1分钟K线通过 BarGenerator 合成）
- **入场逻辑**: 趋势因子穿越阈值（状态改变触发，非每根K线判断）
- **离场逻辑**: 趋势因子回归零轴 + ATR移动止损
- **止损**: 5倍ATR从持仓期间极值回撤

## 参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| ma_short | 10 | 短期MA周期 |
| ma_mid1 | 20 | 中期MA1 |
| ma_mid2 | 50 | 中期MA2 |
| ma_long1 | 100 | 长期MA1 |
| ma_long2 | 200 | 长期MA2 |
| entry_threshold | 0.008 | 入场阈值（价格偏离百分比） |
| exit_threshold | 0.0 | 离场阈值 |
| atr_window | 20 | ATR计算周期 |
| atr_stop_multiplier | 5.0 | ATR止损倍数 |
| fixed_size | 1 | 固定交易手数 |

## 回测结果

### rb99.SHFE（螺纹钢，1小时K线，2016-2026）

| 条件 | Sharpe | 总收益 | 最大回撤 | 交易次数 |
|------|--------|--------|----------|----------|
| slippage=0 | 0.58 | 23.7% | -4.22% | 891 |
| slippage=1 | 0.47 | 19.2% | -4.76% | 891 |
| slippage=2 | 0.36 | 14.7% | -6.31% | 891 |

### 最优品种：j99.DCE（焦炭）

| 指标 | 值 |
|------|-----|
| Sharpe | 0.95 |
| 总收益 | 297.5% |
| 最大回撤 | -19.7% |
| 交易次数 | 1005 |
| 前半段Sharpe | 1.17 |
| 后半段Sharpe | 0.62 |

## 评估

- **优点**: 能承受真实交易成本（slippage=1 仍盈利），多品种适用
- **不足**: Sharpe未达毕业标准（1.0），后半段效应减弱
- **结论**: 保留代码，可用但未达标
