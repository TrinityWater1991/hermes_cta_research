# Trend State Machine 研究项目 — 已关闭

## 来源
Royal Purple Dual Asset Trend Follower — Trading & Investing Strategies (Discole), 2026-05-05

## 核心思路
将趋势跟踪视为状态机：趋势是持续状态而非事件信号。
三层EMA（Fast/Slow/Regime）确认趋势状态，ATR自适应移动止损，被止损后若趋势仍存则重入场。

## 研究方向

### v1: 三层EMA趋势状态机 + ATR移动止损 ✅ 完成
- 使用1小时K线判断趋势状态（BarGenerator从1分钟合成）
- 三层EMA（Fast/Slow/Regime）确认趋势方向
- ATR自适应移动止损，追踪止损
- 趋势反转离场 + 重入场机制（冷却期控制）

### v2: 趋势状态 + 通道突破 ❌ 未测试
### v3: 自适应移动止损变体 ❌ 未测试

## 最终结论：Retain（保留）

### 最优参数（jm99）
- fast_ema=5, slow_ema=13, regime_ema=34
- atr_period=14, atr_multiplier=3.0（默认）
- min_hold_bars=2, reentry_cooldown=1

### 关键发现
1. 三层EMA趋势状态机在1小时K线上有效，但信号有延迟
2. 策略在jm99上表现明显优于其他品种（品种依赖性强）
3. 策略效应在2023年后显著衰减（前半段Sharpe 1.08→后半段0.40）
4. 交易频率合理（~105次/年），非过度交易
5. 趋势反转离场（反手）机制比纯止损离场能更好捕获大趋势

### 毕业判定
- ✗ 不满足毕业标准（slippage=1下Sharpe未达1.0，后半段衰减严重）
- ✓ 保留代码，策略逻辑可靠，jm99上滑点成本可承受
