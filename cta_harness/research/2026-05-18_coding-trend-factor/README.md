# Coding Trend Factor 研究项目

## 来源

文章：[Coding Trend Factor](../raw/articles/2025-02-07_coding-trend-factor.md)

## 核心思路

多周期MA归一化信号的加权组合作为趋势强度指标，应用于单标的期货趋势跟踪。

## 研究方向

1. **多周期MA等权趋势策略** — 选5-7个周期的MA，归一化后等权求和，超过阈值入场
2. **三周期投票策略** — 短(20)/中(60)/长(200)三条MA，多数看多则做多
3. **自适应权重趋势策略** — 用滚动相关性动态调整各MA的权重

## 回测参数

- 合约：rb99.SHFE
- 数据：2016-05-06 ~ 2026-05-08（1分钟线）
- rate=0.0001, size=10, pricetick=1, slippage=0→1, capital=200,000
- K线周期：1小时（BarGenerator合成），避免过度交易

## 设计要点

- 入场基于趋势因子状态改变（穿越阈值），不是每根K线都交易
- 必须包含ATR移动止损
- 关注每笔平均利润 vs 滑点成本（参考slippage-trap教训）
