# 研究方向：体制依赖的趋势跟踪

**状态: 已淘汰**  
**入库日期**: 2026-05-31  
**关闭日期**: 2026-05-31

## 研究方向

1. **波动率体制检测**：用 ATR/历史波动率分位数划分体制（低波=多仓，高波=少仓/空仓）
2. **信号×体制矩阵**：原有 MA 穿越信号判断方向 + 波动率体制决定仓位倍数
3. **简化版**：趋势做多信号 + 波动率>80%分位时仓位减半

## 回测结果

在 BTCUSDT.BINANCE 1h K线（2019-2026）上测试 10+ 变体：

- v1: EMA交叉 + ATR比率体制过滤 → 最优 Sharpe -0.04
- v2: EMA偏离归一化信号 + 体制过滤 → 最优 Sharpe 0.099

所有变体 Sharpe < 0.3。

## 结论

**Discard**。体制感知分配框架在多资产组合层面有效，无法移植为加密单标的CTA策略。关键失败原因：信号本身无alpha + 组合分散化逻辑在单标的上失效。
